# Requisitos Funcionais
- RF1: Opção de cadastro 
- RF2: Opção de login
- RF3: Opção de exclusão de perfil para o usuário.
- RF4: Cabeçalho de Opçoes entre as abas.
- RF5: Cardapio
- RF6: Colocar uma aba para contato
- RF7: Promoçoes aleatorias para diversidade do site
- RF8: Aba que junta todas as promoções
- RF9: Aba para carrinho com as encomendas desejadas
- RF10: Aba em que estão todos gatos

# Requisitos não funcionais
- RNF1: Acessibilidade para multiplataforma
- RNF2: Fazer o Banco de dados em Postgre
- RNF3: O sistema deve ser responsivo 
- RNF4: Alto desempenho
- RNF5: Alta usuabilidade
